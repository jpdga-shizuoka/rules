# ディスクゴルフ競技規則プロジェクト

これはディスクゴルフ競技規則のための公開プロジェクトです.
本プロジェクトの成果は,
[https://jpdga-shizuoka.github.io/rules](https://jpdga-shizuoka.github.io/rules)
にて公開されています.

## プロジェクトの更新

更新内容を
[https://jpdga-shizuoka.github.io/rules](https://jpdga-shizuoka.github.io/rules)
に公開するためには,
以下のコマンドを実行してください.

```
$ ./deploy
```
